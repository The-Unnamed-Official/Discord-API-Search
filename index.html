About

This static page calls /users/{id} from Discord's REST API using a Bot token you supply.

  • Never commit your token.
  • 401/403 = bad/unauthorized token.
  • 404 = User not found.
  • 429 = Rate limited (slow down).
Troubleshooting
  • No network entry → extension or offline.
  • Status 0 → CORS / network blocked (rare); use proxy (see README).
